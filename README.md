# README

Excercise in setting up and deploying Rails apps.

# Installation

### Prerequisites

**[Ruby 2.5.0+](https://www.ruby-lang.org/en/downloads/)**
**Rails 5.2.0+**

### Install

```console
git clone git@github.com:Nazeh/toy-app-rails.git
cd toy-app-rails
```

### Run
```console
rails server
```
